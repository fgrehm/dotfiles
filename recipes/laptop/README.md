# laptop

Random terminal tooling for laptop/baremetal use. Skipped in containers via `.recipeignore`.

## What it does

- Installs [vhs](https://github.com/charmbracelet/vhs) — terminal session recorder
- Installs [ttyd](https://github.com/tsl0922/ttyd) — share terminal over the web (vhs dependency)

## Requirements

- Debian 13 (Trixie) or equivalent Linux
- wget
- Internet access (GitHub releases)
