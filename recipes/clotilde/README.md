# clotilde

[Clotilde](https://github.com/fgrehm/clotilde) Claude Code session manager.

## What it does

- Installs the clotilde binary to `~/.local/bin/clotilde` from GitHub releases

## Requirements

- wget
- Internet access (GitHub releases)

## Notes

Clotilde stores all session state in `.claude/clotilde/` within each project's working tree.
No global config to manage here.
